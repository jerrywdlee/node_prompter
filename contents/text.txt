それでは発表を致します。

本研究のテーマは、抗マラリア薬ファンシダール耐性進化の解明です。
<br/>
マラリアは世界三大感染症の一つとされています。
WHOによると2014年、約2億例のマラリアが報告され、58.4万人が亡くなりました。その多くはアフリカの子供でした。
<br/>
マラリアの病原体マラリア原虫は、多数の種がありますが、ヒトに寄生する主な種はこの４つです。
その中、熱帯熱マラリア原虫及び三日熱マラリア原虫が大半を占めています。
<br/>
マラリア原虫の多くはアフリカに分布していますが、パプアニューギニアなどアジア太平洋地域では熱帯熱マラリア原虫と三日熱マラリア原虫両方分布しています。
<br/>
ファンシダールは抗マラリア薬サルファドキシン及びピリメサミンの合剤で、略してＳＰといいます。
妊娠中のマラリア原虫感染は新生児死亡の頻度を上げるため、SPは妊娠中予防的治療に広く用いられています。
<br/>
マラリア原虫のDNA合成にはテトラヒドロ葉酸(THF)が必要となっています。ＳＰはマラリア原虫のジヒドロ葉酸還元酵素（DHFR）及びジヒドロプテロイン酸合成酵素（DHPS）を阻害することで、THFの合成を阻害します。
<br/>
SPはマラリア原虫のDNA合成を阻害できるので、マラリア原虫のDNA合成一番盛んでいるシゾント期に最も有効とされています。
<br/>
しかし、マラリア原虫はSP耐性を獲得しています。
過去の研究では熱帯熱マラリア原虫のSP耐性は、pfdhfrとpfdhpsの特定部位の変異によって起こります。
<br/>
図が示すように、pfdhfrもpfdhpsも耐性レベルの上昇は突然変異の増加と比例しています。
Pfdhfrコドン108の変異はピリメサミンに対するIC50を100倍にします
Pfdhpsコドン437の変異はサルファドキシンに対するIC50を10倍にします
<br/>
一方、三日熱マラリア原虫においても、熱帯熱マラリア原虫と同様に、pvdhfrとpvdhpsの特定部位における変異がSP耐性に関与することが知られています
<br/>
しかし、三日熱マラリア原虫と熱帯熱マラリア原虫のどちらがSPに耐性化しやすいのかについては、まだ明らかになっていません。
この点を解明するため、パプアニューギニア(PNG)における検体を用いて、分子疫学的な検討をおこないました。
<br/>
PNGにおいては妊産婦の予防的治療の他に、治療薬としてもSPが使われていました。
特に、2000年～2010年にわたり第一選択薬としてSPが使用され、マラリア原虫に強い選択圧を与えてました。
<br/>
PNGは太平洋に面する島国です
ＷＨＯによると、PNGの人口の94％がマラリア高度流行地区に暮らしています。2014年では28万例のマラリアが報告されました。
<br/>
<br/>
本研究は当講座の美田教授たちが2012年二月、PNG・東セピック州の6集落の全村民に実施した疫学調査で得られた血液フィルターペーパー標本737例です。これらの検体はChelexを用いて原虫DNA抽出し、分析を行いました。
<br/>
まず、我々は血液標本を診断ＰＣＲでマラリアの有無を診断し原虫種を同定しました。
<br/>
ご覧のように、当地区で検出されたマラリア原虫の多くは三日熱マラリア原虫でした。
<br/>
地理的位置が異なるARAHAIMとWOGINARA両集落を除いて、感染者が占める割合並びに感染したマラリア原虫種の比例はどの集落も類似していました。
<br/>
そして我々は熱帯熱マラリア原虫のSP耐性関連遺伝子pfdhfr及びpfdhpsに対しNested-PCRを行い、ダイレクトシークエンスを行いました。
その結果、熱帯熱マラリア原虫において、pfdhfr、pfdhpsのハプロタイプとも野生型、単変異、二重変異３つがありました。
<br/>
2012年pfdhfrではすべてのサンプルが二重変異のCNRNIでした。そしてpfdhpsはサンプルの66%が二重変異のSGEAAでした。
<br/>
これら突然変異の分布状況を地図に照合した結果、pfdhfr,pfdhpsとも、どの集落での分布も類似していました。
<br/>
そして2002－３年同地区の調査結果と比較すると、pfdhfrのハプロタイプのなか、二重変異が占める割合は75%から100%まで増えました。Pfdhpsの二重変異ハプロタイプは0%から66%まで増えました。
<br/>
三日熱マラリア原虫のSP耐性関連遺伝子pvdhfr及びpvdhpsに対しても、熱帯熱マラリア原虫と同じように、Nested-PCRを行い、ダイレクトシークエンスを行いました。
その結果、pvdhfrは野生型、単変異、二重変異合計５つハプロタイプがありました。Pvdhpsは野生型、単変異２つありました
<br/>
2012年pvdhfrでは、サンプルの68%は二重変異のCNRTI、野生型はわずか11%でした。Pvdhpsでは、一サンプルを除いて、すべてが野生型でした。
<br/>
2012年pvdhfrの分布は、SOWOMを除いたすべての集落は類似していました。
Pvdhpsは一例を除いて、すべて野生型でした。
<br/>
2002-3年同地区の調査結果と比較すると、pvdhfrは十年間、二重変異のCNRTIが 50%から68%まで増加し、野生型は26%から11%まで減少しました。Pvdhpsの殆ど変化はありませんでした。
<br/>
2002-3年と2012年熱帯熱及び三日熱マラリア原虫のSP耐性遺伝子を比較したところ、2012年三日熱マラリア原虫各ハプロタイプの頻度は2002-3年の熱帯熱マラリア原虫と類似していることがわかりました。そこで、三日熱マラリア原虫薬剤耐性進化は熱帯熱マラリア原虫より約10年遅れていると考えられます。
<br/>
本研究の結果をベースに、考察を3点まとまりました。

考察１、PNGでは三日熱マラリア原虫が優占種になっていました。過去WHO等が実施した調査では、PNGで流行しているマラリア原虫のうち、大半は熱帯熱マラリア原虫でした。
<br/>
その原因について二点考えました。
第一に、マラリア治療が症状を示す血中マラリア原虫に集中しています。症状を示さない三日熱マラリア原虫の休眠体に対する治療はまだ不十分とされています。
第二に、三日熱マラリア原虫の感染者の血中原虫密度はしばしば熱帯熱マラリア原虫より低く、過去の検討では見逃されていた可能性があります。
<br/>
考察２、ピリメサミン耐性はサルファドキシン耐性より先に獲得されています。
考えられる理由として、
まず、マラリア原虫の葉酸回路はde novo回路とsalvage回路2つ存在しています。ピリメサミンは2回路全部阻害できますが、サルファドキシンはde novo回路のみ阻害できます。
ゆえにSPの薬剤プレッシャーの下でピリメサミン耐性を持つマラリア原虫はより拡散しやすいと考えられます。
そして、ピリメサミンの半減期は比較的に長いです。半減期が長いことで、血中ピリメサミン濃度は徐々に下がり、弱い薬剤耐性を持つ原虫株もピックアップされやすいと考えられます。
<br/>
考察３、三日熱マラリア原虫は熱帯熱マラリア原虫より10年程遅れてSP耐性進化が起こっていました。
考えられる理由として、三日熱マラリア原虫のSP感受性は熱帯熱マラリア原虫より低いこと、そして三日熱マラリア原虫の感染は熱帯熱マラリア原虫の感染より症状が軽いので、治療を受ける可能性は比較的低いことがあります。
この２点で、三日熱マラリア原虫がPNGで受けたSPの薬剤プレッシャーは、熱帯熱マラリア原虫より弱い可能性があると考えました。
<br/>
結論として、PNGにおける本検討では、熱帯熱マラリア原虫、三日熱マラリア原虫ともにピリメサミン耐性がサルファドキシン耐性より先に進化していました。また、熱帯熱マラリア原虫のほうが三日熱マラリア原虫よりSP耐性化しやすいこともわかりました。
